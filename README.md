# Analysis of Exchange Rate

  1. 프로젝트 개요
  + 기간
    - 2019 11 12 2019 12 17 약 36 일간
  +주제
    - 16 개국의 15 년치 환율 분석
  +프로젝트목표
    - 연도별 평균 환율 국가별로 몇 년도에 환율이 가장 높았는 지
    - 분기별 평균 환율 국가별로 몇 분기에 가장 환율이 낮았는 지
    - 현찰로 외화를 사고 팔 때 몇 년도에평균 금액 차이가 가장 크게 나는 지
